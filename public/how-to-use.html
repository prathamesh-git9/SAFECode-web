Supported Vulnerabilities

Buffer Overflow (CWE-120)

Detects unsafe string operations like strcpy(), gets(), and sprintf().

Format String Vulnerability (CWE-134)

Identifies printf() calls with user-controlled format strings.

Command Injection (CWE-78)

Detects dangerous system() calls with user input.

Integer Overflow (CWE-190)

Finds potential integer overflow in size calculations.

Example Vulnerable Code

#include <stdio.h>
#include <string.h>

int main() {
    char buffer[10];
    char *user_input = "This is a very long string";
    
    // Vulnerable: Buffer overflow
    strcpy(buffer, user_input);
    
    // Vulnerable: Format string
    printf(user_input);
    
    // Vulnerable: Command injection
    system(user_input);
    
    return 0;
}

Fixed Code Example

#include <stdio.h>
#include <string.h>

int main() {
    char buffer[10];
    char *user_input = "This is a very long string";
    
    // Fixed: Bounded copy
    strncpy(buffer, user_input, sizeof(buffer) - 1);
    buffer[sizeof(buffer) - 1] = '\0';
    
    // Fixed: Safe format string
    printf("%s", user_input);
    
    // Fixed: Removed dangerous system call
    // system(user_input); // SECURITY: Removed for safety
    
    return 0;
}
